=============================
NEAR FUTURE ELECTRICAL v2.0.0
=============================

This pack contains advanced electrical parts - capacitors, nuclear reactors and new battery packs.

============
DEPENDENCIES
============

Required:
- B9PartSwitch (2.18.0)
- Community Resource Pack (1.4.2)
- ModuleManager (4.1.4)
- DynamicBatteryStorage (2.2.4)
- SystemHeat ()

Optional
- Community Tech Tree (v2.2+)

============
INSTALLATION
============

I recommend removing any previous versions of Near Future Electrical before installing an update.

To install, place the GameData folder inside your Kerbal Space Program folder. If asked to overwrite files, do so.

======================
INSTALLATION OF EXTRAS
======================

Some extra patches are bundled that you can use to tweak your installation. To install them, drop the correct folder from the Extras folder in this archive into your KSP GameData Folder

DecayingRTGs: Causes RTGs to lose power generation abilities over time (a few years)

============
LOCALIZATION
============

This mod includes localization support, and includes translations for
- Spanish (KSP forum user fitiales)
- Chinese (KSP forum users DYZBX from the Black Domain Studio(BDS) and tinygroz)
- Russian (KSP forum users Dr. Jet and Sool3)
- Portugese (KSP forum user GabrielGABFonesca)
- German (KSP forum user Three_Pounds)

============
KSP-MINI-AVC
============

This mod is compatible with KSP-AVC version checking. For a comprehensive version checking experience, please download the appropriate plugin (http://forum.kerbalspaceprogram.com/threads/79745).
